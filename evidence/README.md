# Evidence

This directory contains artifacts generated during the execution of Proofora RootVault implementation.

- `tool-versions.txt`: Versions of the tools used in the environment.
- `public-input-manifest.json`: Defines the structure of public inputs for the ZK proof.
- `verifier-artifacts.json`: Contains hashes for the generated proving and verifying artifacts.
- `local-latest.json`: Evidence of successful local proof verification.
- `mutation-results.json`: Output of negative mutation tests.
- `testnet-latest.json`: Testnet deployment details and verification results.
- `testnet-links.md`: Links to the relevant testnet transactions on a block explorer.
- `auditor-receipt.json`: Exported receipt for auditors or judges.
